# مودال موبایل WCLF — راهنمای Elementor

نسخه: **2.9.23**

## کار شما در Elementor

### ۱) تریگرها (۲ آیکن SVG)
روی هر ویجت آیکن / HTML در **Advanced → CSS Classes**:

| کلاس | کار |
|---|---|
| `wclf-open-filters` | باز کردن مودال فیلترها |
| `wclf-open-sorting` | باز کردن مودال مرتب‌سازی |

آیکن‌ها را فقط در موبایل نشان دهید (Responsive Hide on Desktop).

### ۲) منبع محتوا
روی **کانتینر/ستون والد** (نه روی خود شورت‌کد):

| کلاس | محتوا |
|---|---|
| `wclf-mobile-filters-source` | دسته + برند + قیمت |
| `wclf-mobile-sorting-source` | `[beban_product_filters]` |

### ۳) رفتار
- فقط عرض ≤ ۷۶۷px
- یک مودال در هر لحظه
- بستن با × / backdrop / Escape
- بعد از انتخاب فیلتر/سورت، مودال بسته می‌شود تا نتیجه AJAX دیده شود
- دسکتاپ بدون تغییر

## تست سریع
1. موبایل: کلیک آیکن فیلتر → sheet باز شود و فیلترها دیده شوند  
2. انتخاب دسته → sheet بسته + نتایج عوض شوند  
3. آیکن سورت → مودال جدا  
4. دسکتاپ: سایدبار و سورت مثل قبل؛ مودال دیده نشود
